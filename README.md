# Git Command Showcase

## Clone Repository
git clone https://github.com/Gowri-Paithacs/git-command-showcase.git

## Check Status
git status

## Stage Files
git add .

## Commit Changes
git commit -m "message"

## Push Changes
git push

## Pull Latest Changes
git pull

## Create Branch
git checkout -b branch-name

## Git Stash
git stash
